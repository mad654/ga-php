# ga-php
Genetic algorithm implemented based on this tutorial: http://www.ai-junkie.com/ga/intro/gat1.html

## How to run
```
git clone git@github.com:mad654/ga-php.git
cd src
php SimpleDemo.php
```

See [src/SimpleDemo.php](src/SimpleDemo.php) for details.
